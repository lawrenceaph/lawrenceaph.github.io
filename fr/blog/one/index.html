Magnifique Github

  • Parfois, j’aimerais pouvoir simplement apprendre par osmose. Avec Github, j’ai trouvé la meilleure chose suivante.* Photo: Dmitry Ratushny | unsplash

Dans le trou de lapin

J’ai récemment compris comment utiliser les pages Github pour héberger un site de blog créé par Hugo (À cause de ce . En apprenant la méthode, je suis devenu accro à la beauté de Github. Après avoir bricolé et compris un peu plus Github, j’en suis tombé amoureux.

Apprentissage incrémental

Les fonctionnalités de gestion des connaissances de Github sont un support formidable pour le style d’apprentissage incrémentiel vers lequel je suis naturellement attirée. Ce blog est l’un de ces projets d’apprentissage, et j’espère mener de nombreuses expériences ici et ailleurs. Revenez bientôt pour plus de contenu. Si vous faites quelque chose de similaire, faites-le moi savoir dans les commentaires ci-dessous. Acclamations!

Problèmes avec le sous-domaine - corrigé

J’ai oublié que j’avais commenté / désactivé la ligne dans le fichier yml qui spécifiait mon sous-domaine, ce qui a entraîné la suppression constante du sous-domaine de la configuration des pages lors des mises à jour du référentiel public. Cela a maintenant été corrigé. Yay. Et, leçon apprise.
